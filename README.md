# Study Tracker (React + Vite)

A simple study task tracker where you can add tasks, mark them done, edit, delete, and reorder them.
Tasks are stored in a Spring Boot API backed by PostgreSQL.

## Live Demo
- Frontend (Vercel): https://study-tracker-nine-mu.vercel.app/
- Backend API (Render): https://study-tracker-backend-qlkv.onrender.com

## Features
- Create, edit, delete tasks
- Mark tasks done / undone
- Filter tasks (All / Active / Done)
- Reorder tasks (UI)
- Persistent storage via backend API + PostgreSQL
- Vercel Analytics enabled

## Tech Stack
- React + Vite
- JavaScript (Fetch API)
- Deployed on Vercel
